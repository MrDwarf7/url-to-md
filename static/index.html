url2md

Any URL → clean markdown. Strips the junk your LLM doesn't need.

Clean output

                
Readability extraction grabs just the article body. No nav bars, no cookie banners, no "related posts" garbage.

            

            

                
Token counts

                
See exactly how many tokens you're burning before you paste into Claude or Cursor. No more context window surprises.

            

            

                
Free API

                
30 requests/hour, no signup. Curl it, script it, wire it into your CI. It's a public utility, not a walled garden.

            

            

                
500K chars

                
Handles long docs, blog posts, even full changelogs. Most pages land under 20K chars — you've got room.

            

        

        
        

            
API

            
curl "https://url2md.com/api/convert?url=https://example.com"

            
curl -X POST https://url2md.com/api/convert \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com", "max_chars": 50000}'

            

                Response: {url, title, markdown, char_count, token_estimate, line_count}

                Rate limit: 30 requests/hour per IP. No API key needed.
